Tablas de frecuencia

Los datos cuantitativos discretos se organizan en tablas, llamadas Tablas de Distribuci√≥n de frecuencias. La primera columna de la tabla contiene los distintos valores que asume la variable ordenados de menor a mayor y las restantes columnas contienen los siguientes tipos de frecuencias. ‚Ä¢ Frecuencia absoluta: Indica el n√∫mero de veces que se repite un valor de la variable. ‚Ä¢ Frecuencia relativa: Indica la proporci√≥n con que se repite un valor. Se obtiene dividiendo la frecuencia absoluta entre el tama√±o de la muestra. Para una mejor interpretaci√≥n es m√°s conveniente mutiplicarla por 100 para trabajar con una Frecuencia relativa porcentual. ‚Ä¢ Frecuencia absoluta acumulada: Indica el n√∫mero de valores que son menores o iguales que el valor dado. ‚Ä¢ Frecuencia relativa porcentual acumulada: Indica el porcentaje de datos que son menores o iguales que el valor dado.

Gr√°fico de pie

Un gr√°fico de pie es aquel cuya divisi√≥n proporcional busca representar la distribuci√≥n de categor√≠as dentro de una variable nominal u ordinal: el √°rea del c√≠rculo representa el 100% de los casos de una variable, mientras que el √°rea de cada divisi√≥n del c√≠rculo, representa el porcentaje de casos en una categor√≠a espec√≠fica de esa variable (Ritchey 2008, 80‚Äì83)

Distribuci√≥n normal

La distribuci√≥n normal, es una de las distribuciones de probabilidad de variable continua que con m√°s frecuencia aparece en estadistica y tambien en la teoria de probabilidades. Para realizar la distribuci√≥n normal en R, debemos tener en cuenta:

-Al hallar una probabilidad en x exacta, se debe utilizar la funci√≥n dnorm(). -Al hallar una probabilidad en x<=, se debe utilizar la funci√≥n pbinom(). -Al hallar una probabilidad en x>=, se utilizar√° y activar√° la opci√≥n lower.tail y se colocar√° en FALSE, asi tendriamos el x>=pbinom(lower.tail=FALSE)

Por ejemplo:

Para calcular la P(Z>1) de una N(0,1)

pnorm(1, mean = 0, sd = 1, lower.tail = F)

Para calcular la P(-2<Z<2) de una N(0,1)

pnorm(c(2), mean = 0, sd = 1) - pnorm(c(-2), mean = 0, sd = 1)
[1] 0.9544997

Para calcular la P(0<Z<1.96) de una N(0,1)

pnorm(1.96, mean = 0, sd = 1) - pnorm(0, mean = 0, sd = 1)
[1] 0.4750021

Para calcular la P(Z<=z)=0,5793 de una N(0,1)

qnorm(0.5793, mean = 0, sd = 1)

Para calcular la P(Z>150) de una Normal de media 125 y la desviaci√≥n est√°ndar 50.

pnorm(150, mean = 125, sd = 50, lower.tail = F)
